## A. BGT03-F 예산 편성

### A.1 화면 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | BGT03-F |
| URL | `/budgets/establish?year=:y&department_id=:id` |
| 화면 유형 | F (Form) |
| 접근 권한 | SHM+ (편성·결재 요청) |

### A.2 진입 경로

- BGT01-V 부서 행 [편성] 클릭
- BGT02-V 노드 [편성 보기]
- 새 회계연도 시작 시 새올 알림 [편성 시작]

### A.3 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ ← 예산 현황                                                │
│                                                            │
│ 자치행정과 2026 예산 편성                                   │
│ [편성 중] (결재 미요청)                                     │
│                                                            │
│ 부서 합계: 250,000천원                                      │
│                                                            │
│ 예방 항목 트리 (편성 입력)                                   │
│ ─────────────────────────────────────────                  │
│ ▼ 1단계: 안전보건교육                                       │
│   편성액: 50,000 천원                                       │
│   ├─ ▼ 2단계: 정기교육                                      │
│   │   편성액: 30,000 천원                                   │
│   │  ├─ 3단계: 사무직     [20,000] 천원                     │
│   │  └─ 3단계: 현장직     [10,000] 천원                     │
│   └─ 2단계: 신규 채용 교육 [20,000] 천원                     │
│                                                            │
│ ▼ 1단계: 시설 안전 점검                                     │
│   ├─ 2단계: 청사 정기 점검 [40,000] 천원                     │
│   └─ 2단계: 별관 소방 점검 [30,000] 천원                     │
│ ...                                                        │
│                                                            │
│ ⚠ 1단계 합계 ≠ 하위 합계 검증                              │
│   안전보건교육 1단계: 50,000 / 하위 합계: 50,000 ✓          │
│                                                            │
│ 비고                                                        │
│ [_______________________________________________]           │
│                                                            │
│ [임시저장] [결재 요청 (온나라)]                              │
└──────────────────────────────────────────────────────────┘
```

### A.4 입력 항목

각 budget_categories의 3단계 (또는 가능한 가장 하위 단계)에 편성액 입력. 1·2단계는 하위 합계 자동 산출.

| 필드 | 비고 |
|------|------|
| 3단계 항목별 편성액 | 천원 단위. budget_items INSERT/UPDATE |
| 비고 | budgets.description |

### A.5 자동 처리

- 하위 합계 자동 상위 산출
- 부서 합계 = 1단계 합계
- 결재 요청 → 안내 팝업 (RSK02-D §7.1 패턴) → status='REQUESTED' + 새올 알림 (대상자: CEO/GM, category='approval')

### A.6 검증

| 항목 | 검증 |
|------|------|
| 편성액 | 0 이상 정수 |
| 부서 합계 | 0이면 결재 요청 불가 |
| 1단계 합계 = 하위 합계 | 자동 산출이므로 항상 일치 (수동 입력 시 경고) |

## B. BGT04-F 예산 집행

### B.1 화면 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | BGT04-F |
| URL | `/budgets/execute?budget_item_id=:id` 또는 부서·연도 컨텍스트 |
| 화면 유형 | F (Form / Detail) |
| 접근 권한 | SHM+ (집행 입력) |

### B.2 진입 경로

- BGT01-V 부서 행 [집행]
- BGT02-V 노드 [집행 보기]

### B.3 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ ← 예산 현황                                                │
│                                                            │
│ 자치행정과 2026 예산 집행                                   │
│ [결재 완료] 편성: 250,000 / 집행: 180,000 (72%)             │
│                                                            │
│ 집행 항목 선택                                              │
│ 항목                       │편성     │집행     │집행률      │
│ 안전보건교육 - 정기교육 - 사무직│20,000 │14,000  │70%       │
│ 안전보건교육 - 정기교육 - 현장직│10,000 │ 7,500  │75%       │
│ 시설 점검 - 청사 정기 점검    │40,000 │28,000  │70%        │
│ ...                                                        │
│                                                            │
│ [+ 집행 등록]                                              │
│                                                            │
│ 집행 이력 (선택 항목)                                       │
│ ─────────────────────────────────────                      │
│ 일자       │ 금액     │ 거래처     │ 내용          │첨부 │ │
│ 2026-04-15│  8,000   │ A교육원   │ 1Q 정기교육   │ 📎  │ │
│ 2026-03-20│  6,000   │ B인쇄소   │ 교재 인쇄     │ 📎  │ │
│ ...                                                        │
└──────────────────────────────────────────────────────────┘
```

### B.4 입력 항목 (집행 등록 모달)

| 필드 | 타입 | 필수 |
|------|------|------|
| 편성 항목 (선택) | TREE_SELECT (budget_items) | ✅ |
| 집행일자 | DATE | ✅ |
| 집행액 | NUMBER (천원) | ✅ |
| 거래처 | TEXT | ❌ |
| 내용 | TEXTAREA | ✅ |
| 첨부 (영수증 등) | FILE | ❌ |

### B.5 자동 처리

```
[집행 등록] 저장:
  1. budget_executions INSERT
  2. budget_items 집행액 자동 갱신 (View 또는 캐싱)
  3. 집행 초과 검증:
     - budget_executions 합계 > budget_items.amount
     - 초과 시 경고 ("편성액을 초과합니다 - {초과액}원")
     - 결재 정책에 따라 차단 또는 진행 허용 (협의)
  4. 1·2·3단계 집행률 자동 갱신
```

### B.6 검증

| 항목 | 검증 |
|------|------|
| 집행 항목 | 결재 완료(`APPROVED`) 편성만 선택 가능 |
| 집행액 | 0 초과 |
| 집행 초과 | 경고 (차단 정책 협의) |
