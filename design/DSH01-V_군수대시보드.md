## 4. 표시 항목 (위젯 단위)

### 4.1 종합 KPI 위젯

| 지표 | 데이터 소스 | 산출 방식 |
|------|------------|---------|
| 의무이행률 | `compliance_executions` (Phase 3) | 완료 / 전체 × 100 (올해 누적) |
| 위험성평가 진행률 | `risk_assessments` | `cycle_stage='DONE'` / 전체 × 100 (올해) |
| 결재 대기 | `risk_assessments`, `contracts`, `policies`, `budgets`, `evaluations` 등 | `approval_status='REQUESTED'` COUNT (전체 모듈) |
| 중대재해 발생 | `incidents` (Phase 3) | 중대재해 COUNT (올해) |

### 4.2 시설 기본현황 위젯

TGT01-V §4.1 요약 카드 동일 (CEO 권한 — 전체 시설 카운트).

### 4.3 결재 대기 위젯

```
SELECT 종합 (UNION):
  risk_assessments WHERE approval_status='REQUESTED'
  contracts WHERE status='APPROVAL_REQUESTED'
  policies WHERE approval_status='REQUESTED' (Phase 2)
  budgets WHERE approval_status='REQUESTED' (Phase 2)
  evaluations WHERE approval_status='REQUESTED' (Phase 2)

표시: 카테고리 배지·제목·요청일 + 클릭 시 해당 상세 화면 이동
```

### 4.4 시기도래 업무 위젯 (결정사항 #15 — 안전NOW 기본 색상)

```
임계값·색상:
  D-1개월: 회색
  D-1주:   노랑
  D-당일:  주황
  D+(초과): 빨강

소스 통합: 정기 위험성평가, 정기 안전점검, 안전보건교육, 이행 항목 등
표시: 임계값 카운트 카드 + 임박 5건 목록
```

### 4.5 부서별 의무이행 현황 위젯

```
모든 부서의 의무이행률 (compliance_executions GROUP BY department_id)
막대 그래프 + 부서명 라벨 + 부서 클릭 시 GM 대시보드 view 이동 (CEO 권한)
```

### 4.6 산업안전보건법 의무 이행 위젯 (4개 카드)

| 항목 | 데이터 소스 |
|------|------------|
| 안전·보건 인력 충족 | `dedicated_personnel` 충족 부서 / 전체 |
| 위험성평가 시행 | `risk_assessments` 완료 / 등록 (올해) |
| 도급사업 평가 | `contracts` 적격 / 전체 (올해) |
| 종사자 의견청취 | `opinions` 처리 (이번 분기) |

### 4.7 안전 사고·아차사고 추이

`incidents` 시계열 그래프 (Phase 3 모듈 — 사고 등록 SFR-003에서 데이터 발생).

### 4.8 알림 메시지 내력

NTF02-D 드롭다운과 동일 데이터. 본 위젯에서는 위젯 내에 직접 표시.

## 5. 버튼·링크 액션

| 요소 | 액션 |
|------|------|
| KPI 카드 클릭 | 해당 영역 상세 (예: 결재 대기 → 결재 대기 모음 화면) |
| 결재 대기 행 클릭 | 해당 상세 화면 이동 (RSK02-D / CON01-D / POL02-D 등) |
| 결재 대기 [전체 보기] | 결재 대기 모음 화면 (선택 — Phase 2/3 진행 시 결정) |
| 시기도래 행 클릭 | 해당 항목 상세 (RSK02-D / CMP02-D 등) |
| 부서별 그래프 막대 클릭 | 해당 부서 상세 (CEO는 GM 대시보드 view로 진입 가능) |
| 의무 이행 카드 클릭 | 해당 SFR 화면 이동 (STF04-V / RSK02-L / CON01-L / OPN01-L) |
| 알림 행 클릭 | NTF02-D와 동일 — `link_url` 이동 |

## 6. 자동 처리 로직

본 화면은 종합 표시 화면. 직접 트리거 없음. 다음 외부 자동 작업의 결과가 반영:

- 위험성평가 자동 생성 / 결재 상태 변경
- 도급계약 진행 단계 변경 / 안전관리자 미달 차단
- 새 알림 적재 (실시간 폴링 30초)
- 사고 등록·처리
