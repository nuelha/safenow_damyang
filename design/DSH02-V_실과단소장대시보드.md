## 4. 표시 항목 (위젯)

### 4.1 부서 KPI 위젯

GM의 데이터 스코프 = 본 부서 + 산하 모든 데이터.

| 지표 | 산출 |
|------|------|
| 의무이행률 (부서) | 본 부서 산하 완료 / 전체 |
| 위험성평가 진행률 | 부서 산하 관리대상의 평가 완료 / 전체 |
| 결재 요청·확인 | 본인이 결재해야 할 건 + 부서 산하 진행 중 결재 |
| 미할당 안전관리자 | 부서 산하 FACILITY 중 SAFETY_MGR 미지정 |

### 4.2 부서 관리대상 위젯

TGT01-V 요약 카드를 부서 스코프로 적용.

### 4.3 결재 요청·확인 위젯

```
GM에게 결재 요청된 항목 + 본 부서에서 결재 진행 중인 항목:
  - 정책상 CEO 결재 항목은 GM이 검토만 (의견 제출 가능)
  - GM 결재 항목 (있다면 — Phase 2 결정사항)
표시: 우선순위(빨강/주황) + 클릭 시 상세 이동
```

### 4.4 인력 충족 현황 위젯

STF04-V 부서 카드 동일 (안전관리자/보건관리자/관리감독자/산업보건의).

### 4.5 시기도래 업무 위젯

DSH01-V §4.4와 동일하나 부서 스코프 한정.

### 4.6 도급사업 현황 위젯

CON01-L 요약 카드를 부서 스코프로 적용.

### 4.7 부서 개선조치 현황 위젯

`improvements WHERE department_id` 집계 (진행 중·기한초과·완료).

### 4.8 부서 사고·아차사고

DSH01-V §4.7과 동일하나 부서 한정.

### 4.9 알림 메시지

NTF02-D 동일.
