## 4. 표시 항목

### 4.1 목록 컬럼

| 컬럼 | 데이터 소스 | 비고 |
|------|------------|------|
| 출처 배지 | `users.source_type` | SSO / 직접 / 외부 (배지 색상 매핑 §4.2) |
| 이름 | `users.name` | 클릭 시 ORG02-D |
| 부서 | JOIN `departments.name` | LOCAL 사용자는 `external_company` 표시 |
| 직급 | `users.position` | |
| 권한 | `users.permission_role` 배지 | CEO/GM/SM/SHM/WKR/SUB/CON |
| 자격증 건수 | `COUNT(user_qualifications WHERE user_id=:u.id)` | 클릭 시 ORG02-D 자격증 탭 |
| 관리 | — | [상세] 버튼 |

### 4.2 출처 배지

| source_type | 배지 | 색상 |
|-------------|------|------|
| PORTAL_SYNCED | SSO | 파랑 |
| MANUAL | 직접 | 회색 |
| LOCAL_INVITED | 외부 | 주황 |

## 7. 자동 처리 로직

본 화면 자체 트리거 없음. 다음 외부 자동 작업이 본 화면에 반영:

- 행정포털 조직도 정기 동기화 (IF-002): SSO 사용자 자동 UPSERT
- ORG02-M 직접 등록 시 즉시 목록 갱신
- ORG02-M2 일괄 등록 시 신규 N건 추가 후 목록 갱신

## 8. 데이터 스코프

| 권한 | 보이는 범위 |
|------|----------|
| CEO | 전체 |
| GM | 본 부서 + 산하 부서 소속 사용자 |
| SM/SHM | 본 부서 소속 + 본인 담당 관리대상 관련 사용자 |
| WKR | 본인 정보만 (목록 접근 시 본인 단일 행) |
| SUB/CON | 본 화면 접근 불가 |

## 9. 오류 처리

| 케이스 | 메시지 | 처리 |
|--------|--------|------|

| 권한 부족 (외부 접근) | "접근 권한이 없습니다" | AUTH99-V |
