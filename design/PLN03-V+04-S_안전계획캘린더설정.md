### A.4 표시 항목

| 항목 | 데이터 소스 |
|------|------------|
| 캘린더 이벤트 | `plan_items.due_date` (또는 scheduled_month) 기준 |
| 카테고리 색상 | category별 색상 매핑 |
| 이벤트 라벨 | `plan_items.title` (truncate) |
| 부서·관리대상 필터 | 스코프 적용 |

### A.5 뷰 모드

| 뷰 | 표시 단위 |
|------|---------|
| 월 (기본) | 월별 그리드 + 일별 이벤트 점 |
| 주 | 주별 타임라인 + 시간 슬롯 |
| 일 | 일별 시간순 목록 |

### A.6 버튼·링크 액션

| 요소 | 액션 |
|------|------|
| `[< / >]` 월 이동 | 이전/다음 월 |
| `[월/주/일]` 뷰 토글 | URL `?view=month/week/day` |
| 필터 변경 | URL 동기화 + 이벤트 재로드 |
| 이벤트 클릭 | 항목 상세 팝오버 + [상세 보기] → 카테고리별 모듈 진입 |
| 일자 클릭 | 해당 일 모든 이벤트 모달 |

### A.7 데이터 스코프

권한별 필터 적용 (CEO=전체 / GM=부서 / SM·SHM=담당 / WKR=소속).

### B.2 화면 내부 서브 탭 구성

> 본 화면은 3개 영역으로 구성 — 페이지 내부 서브 탭으로 전환 (모듈 화면 간 이동과 무관 / 같은 화면 안의 컨텍스트 전환).

```
[표준 템플릿] [카테고리 마스터] [필수 항목 정책]
```

### B.3 표준 템플릿 영역

```
관리대상 type별 / 중대재해법 분류별 안전계획 표준 템플릿 관리.

┌────────────────────────────────────────────┐
│ 표준 템플릿 목록                            │
│                                            │
│ 코드                  │적용 type   │항목 수│관리│
│ FACILITY_PUBLIC_USE  │시설/공중이용│ 12    │[편집]│
│ FACILITY_RAW_MATERIAL│시설/원료    │ 10    │[편집]│
│ BUSINESS             │사업         │ 8     │[편집]│
│ TASK                 │업무         │ 6     │[편집]│
│                                            │
│                            [+ 템플릿 추가]  │
└────────────────────────────────────────────┘

템플릿 편집 모달:
  - 표준 항목 목록 (카테고리·제목·기본 마감월·기본 담당 역할)
  - 항목 추가·수정·삭제·정렬
```

### B.4 카테고리 마스터 영역

`plan_categories` 마스터 관리 (교육·점검·평가·예산·도급·기타). 코드·표시명·색상·기본 정렬 순서.

### B.5 필수 항목 정책 영역

법령상 필수 항목 정의:
- 안전보건교육 (분기 1회 이상)
- 위험성평가 (연 1회 이상)
- 정기 안전점검 (월 1회 이상 또는 분기 1회)
- ...

설정 후 PLN02-F 신규 등록 시 자동 검증 (필수 항목 미포함 시 경고).

### B.6 버튼·링크 액션

| 요소 | 액션 |
|------|------|
| 화면 내부 서브 탭 전환 | URL `?section=` 동기화 |
| 템플릿 행 [편집] | 템플릿 편집 모달 |
| [+ 템플릿 추가] | 신규 템플릿 모달 |
| 카테고리 [편집] | 카테고리 편집 모달 |
| 필수 항목 정책 [편집] | 정책 편집 모달 (CEO만) |

### B.7 데이터 스코프

CEO/GM/SHM 모두 접근 가능. 정책 변경은 CEO만.

## 공통 — 관련 DB 테이블

- `plan_templates` (Phase 2 — 본 문서에서 정의):

```sql
CREATE TABLE plan_templates (
  id              UUID PRIMARY KEY,
  code            VARCHAR(50) UNIQUE NOT NULL,
  target_type     ENUM('FACILITY','TASK','BUSINESS'),
  cdpa_category   ENUM('PUBLIC_USE','RAW_MATERIAL','PUBLIC_TRANSPORT') NULL,
  title           VARCHAR(200) NOT NULL,
  template_items  JSON NOT NULL,   -- 표준 항목 배열
  is_active       BOOLEAN DEFAULT true,
  created_at      TIMESTAMP DEFAULT NOW()
);
```
