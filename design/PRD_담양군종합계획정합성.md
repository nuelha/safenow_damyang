## 0. 작업 개요

### 0.1 목적

담양군에서 군수 결재(2026.2.3)로 시행 중인 **「2026 담양군 중대재해 예방 종합계획」**의 실제 운영 체계·조직·시설·프로세스를 프로토타입과 산출물에 반영하여 정합성을 확보한다.

### 0.2 핵심 출처

- 결재 문서: 재난안전과-2026 / 2026.2.3 / 군수 결재 완료 / 대국민공개
- 법적 근거: 중대재해처벌법 제4조·제5조·제9조
- 시행 주체: 담양군 재난안전과 중대재해팀

### 0.3 작업 범위

- 프로토타입 콘텐츠 데이터 정합화 (조직·시설·인력 마스터)
- 권한 매핑 규칙 갱신 (실제 직급 → 시스템 권한)
- 알림·결재 라인 정합화 (5단계 보고 체계)
- 화면별 표시 항목 검증 (실제 운영 양식·체크리스트 반영)

### 0.4 변경하지 않는 것

- 화면 ID·URL·DB 스키마·SFR 매핑: 변경 없음
- IA 그룹 구조 (8개 GNB): PRD-002에서 정의된 구조 유지
- ORG → STF 이전 (PRD-001 결과): 유지

### 1.1 5단계 조직 체계 (문서 §Ⅱ)

```
[1단계] 군수 — 경영책임자 + 안전보건관리책임자
   ↓
[2단계] 부군수 — 분임 안전보건관리책임자
   ↓
[3단계] 전담조직: 재난안전과 중대재해팀

   ├─ 행정과 (인력)
   └─ 재난안전과장 (재난예방·대응 총괄)
   ↓
[4단계] 관리감독자 — 실과단소장 + 읍면장 (39개 부서)
   ↓
[5단계] 분임관리감독자 — 팀장 → 종사자
```

### 1.12 안전보건경영방침 7원칙 (문서 붙임1)

```
1. 모든 경영활동에 있어 임직원의 안전을 최우선 가치로 한다.
2. 안전보건기준과 법규를 준수하여 산업재해를 예방한다.
3. 근로자의 안전보건활동을 보장하고 적극적으로 반영한다.
4. 선제적 안전보건관리를 통해 안전하고 쾌적한 사업장을 조성한다.
5. 모든 임직원은 적극적인 참여의식을 통해 안전문화를 정착시킨다.
6. 안전보건교육을 통해 안전의식을 향상하고 산업재해 예방에 앞장선다.
7. 先안전 확보, 後작업으로 작업장 무사고, 무재해 추구
```
