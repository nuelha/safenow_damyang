## 4. 표시 항목

### 4.1 목록 컬럼

| 컬럼 | 데이터 소스 | 비고 |
|------|------------|------|
| 평가명 | `risk_assessments.name` | 클릭 시 RSK02-D |
| 관리대상 | JOIN `targets.name` + type 배지 | |
| 유형 | `assessment_type` 배지 | 정기 / 최초 / 수시 |
| 시행확인 라벨 | `confirmation_label` | "최초" / "수시" (정기는 미표시) |
| 상태 | `cycle_stage` 배지 | 사전준비 / 진행중 / 완료 |
| 기간 | `start_date ~ end_date` | |
| 담당자 | JOIN `users.name` (owner_user_id) | |
| 트리거 | `trigger_type` 배지 | 수시일 때만 (사고/민원/중대변경/기타) |
| 출처 | `source` 배지 (선택) | 수동 / 차세대 e호조 자동 / 정기 자동 / 최초 자동 |
| 결재 상태 | `approval_status` 배지 | 미요청 / 결재 진행중 / 완료 / 반려 |

### 4.2 유형 배지

| assessment_type | 표시명 | 색상 |
|----------------|--------|------|
| `initial` | 최초 | 회색 |
| `regular` | 정기 | 파랑 |
| `special` | 수시 | 보라 |

### 4.3 진행상태 배지

| cycle_stage | 표시명 | 색상 |
|-------------|--------|------|
| `PREPARING` | 사전준비 | 회색 |
| `IN_PROGRESS` | 진행중 | 파랑 |
| `DONE` | 완료 | 초록 |

## 6. 버튼·링크 액션

> 모듈 내 화면 이동은 좌측 사이드 메뉴(LNB)에서 처리 — 본 표에는 포함하지 않음.

| 요소 | 위치 | 액션 | 조건·비고 |
|------|------|------|---------|
| 필터 셋 | 필터 바 | 각 컬럼별 필터 + URL 동기화 | |
| `[+ 평가 등록]` | 우상단 | RSK02-F 이동 (`/risk-assessment/new`) | SHM+ |
| 행 클릭 | 목록 | RSK02-D 이동 (`/risk-assessment/:id`) | |
| 페이지 번호 | 페이지네이션 | 페이지 이동 (URL `?page=`) | |

## 7. 자동 처리 로직

본 화면 자체 트리거 없음. 다음 외부 자동 작업이 본 화면에 반영:

- **정기 평가 자동 생성 배치** (DB-001 §9.4): 새 평가 자동 INSERT (source='REGULAR_AUTO')
- **차세대 e호조 동기화** (IF-004 §4.6): 도급공사 신규 시 평가 자동 생성 (source='EHOJO_AUTO')
- **관리대상 신규 등록 시** (DB-001 §9.2 연계): 최초 평가 자동 생성 (source='INITIAL_AUTO')

> 자동 생성된 평가는 `source` 배지로 구분 가능. 운영자는 본 화면에서 검토 후 RSK02-D에서 진행.
