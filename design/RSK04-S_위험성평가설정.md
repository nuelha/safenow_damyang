## 7. 자동 처리 로직

> e호조 연동 자동 처리(도급공사 자동 위험성평가 생성 등)는 **SYS01-S §6.4**에서 관리.

본 화면은 콘텐츠 마스터 변경에 따른 자동 처리만:

```
평가 주기 변경:
  → risk_assessment_intervals UPDATE
  → 다음 정기 평가 자동 생성 배치(매일 00:00)에서 즉시 반영
  → 진행 중 평가에는 영향 없음 (역사 보존)

평가 기법 비활성화:
  → 사용 중 평가는 영향 없음 (기존 method_id 유지)
  → RSK02-F 신규 등록 시 선택 불가

위험요인 마스터 삭제·비활성화:
  → 사용 중이면 차단 (비활성만 가능)
  → RSK02-F 신규 등록 시 선택 불가
```

## 11. 관련 DB 테이블

- `risk_assessment_intervals` (DB-001 §4.3): 정기 평가 주기 마스터
- `risk_assessment_methods` (Phase 2 마스터): 평가 기법
- `risk_factor_templates` (Phase 2 마스터): 위험요인 마스터
- `audit_logs` (DB-001 §2.12): 설정 변경 감사
