# STF01-V · STF02-V · STF03-S — 안전관리자 인력 (전체·공사별·설정)

> 화면 ID: **STF01-V** (전체 인력 현황) / **STF02-V** (공사별 선임 현황) / **STF03-S** (법정 인원 마스터)
> 모듈: STF (안전관리자 인력)
> SFR: SFR-010
> 버전: v1.0 | 작성일: 2026-05-11
> 관련 문서: SRS-001 §3 SFR-010, 결정사항 #21 (안전관리자 미달 결재 차단)

> ⚠ STF는 ORG03-V(부서 단위 전담인력)와 보완 관계. ORG는 부서·자격증 중심, STF는 **공사별 + 법정 인원 산출** 중심.


### B.4 표시 항목

| 컬럼 | 데이터 소스 |
|------|------------|
| 공사명 | JOIN `contracts.name` (`contract_type='CONSTRUCTION'`) |
| 예산 | `contracts.budget` (억 단위) |
| 법정 인원 | 공사 규모별 법정 (STF03-S 마스터 기준) |
| 배치 | 해당 공사 관련 SAFETY_MGR 카운트 |
| 충족 | 배치 ≥ 법정 |
| 결재 | `contracts.status` (APPROVED/REJECTED/BLOCKED 등) |

### B.5 안전관리자 미달 → 결재 차단 (결정사항 #21)

```
contracts.status='BLOCKED' 자동 전환 조건:
  - 공사 예산 50억 이상
  - SAFETY_MGR 배치 < 법정 인원

전환 후:
  - CON06-F 결재 진행 차단
  - 본 화면 [결재] 컬럼에 "차단" 표시
  - [안전관리자 선임] 버튼 노출 → ORG03-M 모달 진입

복귀 (안전관리자 추가 선임):
  - contracts.status BLOCKED → EVALUATING (또는 이전 상태)
  - CON 모듈 결재 진행 가능
```

### B.6 차세대 e호조 연동

```
e호조 도급사업 import 시:
  - contracts INSERT (source='EHOJO_AUTO')
  - 본 화면에 자동 노출
  - 안전관리자 검증 자동 실행
```

### B.7 버튼·링크 액션

| 요소 | 액션 |
|------|------|
| 행 [상세] | CON01-D 이동 |
| 행 [안전관리자 선임] | ORG03-M 모달 또는 ORG03-V 진입 |
| `[+ 안전관리자 선임]` | 공사 선택 모달 → 선임 진행 |

### C.7 관련 DB 테이블

```sql
-- 부서 단위 법정 인원 마스터
CREATE TABLE legal_personnel_dept (
  id              UUID PRIMARY KEY,
  min_headcount   INTEGER NOT NULL,
  max_headcount   INTEGER,                              -- NULL=무한
  safety_mgr_required INTEGER NOT NULL DEFAULT 0,
  health_mgr_required INTEGER NOT NULL DEFAULT 0,
  health_doctor_required INTEGER NOT NULL DEFAULT 0,
  is_active       BOOLEAN DEFAULT true
);

-- 공사 규모 법정 인원 마스터
CREATE TABLE legal_personnel_construction (
  id              UUID PRIMARY KEY,
  min_budget      BIGINT NOT NULL,                       -- 원 단위
  max_budget      BIGINT,                                 -- NULL=무한
  safety_mgr_required INTEGER NOT NULL DEFAULT 0,
  is_active       BOOLEAN DEFAULT true
);

-- 자격증 매핑
CREATE TABLE qualification_mapping (
  id              UUID PRIMARY KEY,
  role_type       VARCHAR(50) NOT NULL,                  -- SAFETY_MGR / HEALTH_MGR / ...
  qualification_name VARCHAR(200) NOT NULL,
  qualification_type VARCHAR(50),                         -- 국가기술 / 국가전문 / 민간
  is_active       BOOLEAN DEFAULT true
);
```

## 공통 — 관련 화면 흐름

