## 1-4. 경영방침 점검표 탭 변경사항
- 점검결과 입력 유형 3종 지원:
  - O/X: 라디오 버튼 (O | X)

  - 점수: 숫자 입력 / 100
- 각 항목 옆에 [관련근거 +] 버튼 추가
  → 클릭 시 관련근거 선택 모달 (관련근거·법령 탭 데이터에서 선택)
- 샘플 점검항목 3개:
  1. 안전보건목표 설정 여부 (O/X) | 관련근거: 중대재해처벌법 제4조
  2. 경영방침 문서화 및 공표 여부 (텍스트)
  3. 목표 달성률 검토 (점수)
